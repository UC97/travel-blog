# 🌍 Travel Blog - DevOps Learning Project

## 📌 Overview
A full-stack travel blog application built to learn end-to-end DevOps practices.

## 🏗️ Tech Stack
- Frontend: React
- Backend: Node.js (Express)
- Storage: JSON File (data.json)
- Process Manager: PM2

## 🚀 Features
- View travel blogs
- Add blogs via API
- Persistent storage
- Backend managed using PM2

## ⚙️ Setup Instructions

### Backend
cd backend
npm install
node server.js

### Frontend
cd frontend
npm install
npm start

## 🧪 API Endpoints

### Get Posts
GET /api/posts

### Create Post
POST /api/posts

- Phase 1: Manual Deployment ✅
